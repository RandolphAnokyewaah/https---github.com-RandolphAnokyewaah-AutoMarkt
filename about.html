About AutoTrace Systems

AutoTrace Systems is a fictional company created for educational use in web development and security projects. We simulate what a modern telematics platform could look like, without collecting or storing any real data on this demo site.

What we “do”

In this demo, AutoTrace provides a Share Car Locatio feature. It shows how:

  • Web apps ask for geolocation in a transparent way.
  • User consent is required before location is shared.
  • Location can be sent to back-end services (e.g. Email, SMS).
  • UI and wording can build trust around sensitive data.

Educational goals

  • Practice frontend development (HTML, CSS, JavaScript).
  • Work with third-party APIs like EmailJS.
  • Understand privacy, consent, and legal disclaimers.
  • Design a professional-looking SaaS-style interface.

Important note

This is not a real commercial service. No production tracking, no real customers, no real fleet. It’s a safe sandbox for students and developers to experiment with car-related location features.
